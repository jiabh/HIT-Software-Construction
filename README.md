# 哈工大2021春软件构造实验

## 仓库结构概述

仓库共有4个分支：

- [lab1](https://github.com/jiabh/HIT-Software-Construction/tree/lab1)
- [lab2](https://github.com/jiabh/HIT-Software-Construction/tree/lab2)
- [lab3](https://github.com/jiabh/HIT-Software-Construction/tree/lab3)
- [master](https://github.com/jiabh/HIT-Software-Construction/tree/master)

前三个分支分别对应三个实验，每个实验的简介和全部代码都在各自分支内；`master`分支（当前分支）没有实际的功能，仅用于展示本README和方便Git新手查看代码，详情见下一节。

## 如何克隆仓库

如果你不熟悉Git分支的相关操作，则可以使用如下指令：

```sh
git clone --recursive https://github.com/jiabh/HIT-Software-Construction
```

全部代码将出现在`HIT-Software-Construction/`文件夹中。

如果你对Git稍有了解，则个人更加推荐以下的方式：

```sh
git clone https://github.com/jiabh/HIT-Software-Construction
```

这样不会在master分支上下载重复的代码。对于某次实验，只需要切换到目标分支上即可查看。

如果你只想简单地下载某次实验的最终代码，也可以使用GitHub提供的`Download Zip`功能直接下载。记得下载前利用左上角的按钮切换到目标实验的分支。
